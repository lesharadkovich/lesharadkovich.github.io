Всем привет.

Во первых надо сегодня завершить все задания по SQL которые у вас висят.

Второе: сегодня презентация будет записанная
План на сегодня:
1) Ознакомиться и разобраться с планами запросов - https://www.youtube.com/watch?v=4yHc8-Idf-w&feature=youtu.be

2) Посмотреть что есть в T-SQL. Помните мы рассматривали SQL с вами а сейчас посмотрим на расширенный язык T-SQL который работает в SQL Server. Пока смотрите планы запросов я буду редактировать этот пост добавлять ссылки что надо будет почитать и разобрать.
https://msdn.microsoft.com/en-us/library/ff848809(v=sql.105).aspx
https://docs.microsoft.com/en-us/sql/t-sql/queries/select-over-clause-transact-sql
https://msdn.microsoft.com/en-us/library/ms174290(v=sql.105).aspx

3) Разобраться что такое и как создавать Stored Procedures. Добавлю ссылку сюда.

4) Задание: написать CRUD операций (4 хранимые процедуры) для работы с Order-ами. a) Создать хранимую процедуру которая принимает OrderId возвращает Order и Order Details двумя разными селектами. (GetOrderById) б) Создать хранимую процедуру которая принимает OrderID удаляет Order и Order Details по этому ID. (DeleteOrderById) в) Создать хранимую процедуру которая принимает в себя информацию о Order и массив Order Details и создает новый Order в системе. (CreateOrder) г) Создать хранимую процедуру которая принимает в себя Order ID и информацию о Order и массив Order Details и обновляет Order и его Order Details в системе. Давайте постараемся завтра до обеда SQL закончить. Задания ваши должны быть на 95% проверены. Смотрите коментарии и фиксайте проблемы. Пока будете изучать я буду проверять ваши задания целый день. Если будут какие то вопросы то пишите.
